# Mindful Cognitive Therapy App

## Overview
The Mindful Cognitive Therapy App is designed to help users improve their mental health through mindfulness and cognitive behavioral techniques. It provides guided exercises, tracks progress, and offers resources for users to engage with their mental well-being.

## Features
- **Guided Exercises**: Users can access a variety of exercises aimed at improving mindfulness and cognitive skills.
- **Progress Tracker**: Allows users to track their journey and improvements over time.
- **Resource Library**: A collection of articles and videos on mental health topics.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/tufinoamildreta2023cetis49-bot/Detallito.git
   ```
2. Change into the project directory:
   ```bash
   cd Detallito
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

## Usage
To start the app, run:
```bash
npm start
```
